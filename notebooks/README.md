# Notebooks

Esta carpeta contiene los notebooks de Google Colab que se han desarrollado en las distintas fases del pipeline de OCR, preprocesamiento, comparación textual y análisis de grafos.

## Cuadernos incluidos

| Notebooks | Descripción |
|----------|-------------|
| 01_grafos_edge_gephi.ipynb | Generación de los archivos de aristas utilizados en Gephi. |
| EdgeDefin.ipynb | Definición de las relaciones entre términos para la construcción de grafos. |
| EdgeLibComp.ipynb | Comparación entre versiones y preparación de las estructuras de datos. |
| SeparCap.ipynb | Segmentación automática de las obras en capítulos o unidades de análisis. |
| WER Prueba.ipynb | Evaluación de la calidad del OCR mediante la métrica Word Error Rate (WER). |
