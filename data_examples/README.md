# Ejemplos de datos

En esta carpeta se reúnen algunos ejemplos de los archivos de entrada y salida utilizados durante el desarrollo del pipeline metodológico.

## Contenido

Se incluyen ejemplos representativos de:

- Archivos de entrada utilizados en las distintas fases del procesamiento.
- Archivos de salida generados por el pipeline.
- Muestras de datos empleadas para ilustrar el funcionamiento de los cuadernos de trabajo.

## Derechos de autor

Por motivos de propiedad intelectual, el corpus completo de las obras analizadas no se distribuye públicamente. Se incluyen solo ejemplos que tengan únicamente una finalidad metodológica para la comprensión de la estructura de los datos y del funcionamiento del pipeline.
