# Documentación

En esta carpeta se reúne la documentación complementaria asociada al desarrollo del Trabajo Fin de Máster.

## Contenido

Esta carpeta incorporará progresivamente:

- Esquemas del pipeline metodológico.
- Documentación técnica del proyecto.
- Materiales auxiliares empleados durante el desarrollo de la investigación.
- Recursos necesarios para facilitar la comprensión y reproducibilidad del trabajo.

## Finalidad

La documentación incluida tiene como objetivo complementar los cuadernos de trabajo y los materiales de análisis,que se ven en las otras carpetas, para proporcionar una visión global del proceso metodológico seguido durante el trabajo.
