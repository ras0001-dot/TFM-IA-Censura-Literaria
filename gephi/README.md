# Gephi

En esta carpeta se reúnen los materiales que provienen del análisis de grafos que se ha relizado con Gephi durante el proceso del Trabajo Fin de Máster.

## Contenido

En esta carpeta se incorporarán progresivamente:

- Proyectos de Gephi (`.gephi`).
- Grafos exportados (`.gexf`).
- Imágenes de las redes analizadas.
- Métricas y resultados obtenidos mediante Gephi.

## Finalidad

Gracias a estos materiales se pude documentar la fase de análisis de redes del pipeline, con esto se puede facilitar la comprensión de las representaciones gráficas y métricas empleadas en la investigación.
